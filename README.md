# Restaurant Landing Page

A sophisticated, responsive restaurant landing page with advanced CSS animations, Bootstrap responsive design, and interactive features.

## Features Implemented

### ✨ Animated Food Cards
- Scale and rotation animations on hover
- Smooth transitions with brightness effects
- Cards lift up with shadow enhancement
- Staggered animation delays for visual appeal

### 📱 Responsive Menu Section
- Filter menu items by category (Appetizers, Mains, Desserts, Beverages)
- Smooth fade in/out animations when filtering
- Fully responsive grid layout (1-3 columns based on screen size)
- Mobile-optimized spacing and sizing

### 🎠 Testimonials Carousel
- Auto-rotating carousel with 5-second intervals
- Previous/Next navigation buttons
- Indicator dots for slide selection
- Pause on hover functionality
- Animated star ratings
- Responsive design

### 🎯 Hover Effects
- Navigation links with underline animation
- Buttons with shimmer effect
- Card lift-up effects
- Image zoom and filter effects
- Shadow enhancements on hover
- Smooth color transitions

### 📲 Mobile-Friendly Layout
- Fully responsive design (mobile-first approach)
- Breakpoints: 768px (tablet), 576px (mobile)
- Adjustable font sizes for different devices
- Mobile navigation collapse
- Touch-friendly button sizes

## Sections Included

1. **Navigation Bar**
   - Fixed sticky navbar with smooth scrolling
   - Mobile hamburger menu
   - Link hover animations

2. **Hero Section**
   - Full-screen background image
   - Animated heading and subtitle
   - Call-to-action buttons
   - Parallax scroll effect

3. **Menu Section**
   - 12 menu items across 4 categories
   - Filter functionality
   - Card animations
   - Price display

4. **Chef's Specials**
   - 3 featured items with badges
   - Limited time indicators
   - Special pricing
   - Order buttons

5. **Testimonials**
   - Carousel with customer reviews
   - 5-star ratings
   - Review statistics
   - Auto-scroll feature

6. **Reservation Section**
   - Complete booking form
   - Form validation
   - Contact information
   - Restaurant details (hours, location, phone, email)

7. **Footer**
   - About section
   - Quick links
   - Social media links
   - Copyright information

## How to Use

1. Open `index.html` in a web browser
2. Navigate through sections using the navbar
3. Hover over menu items to see animations
4. Use the menu filters to view items by category
5. Browse customer testimonials in the carousel
6. Fill out the reservation form to book a table

## Responsive Breakpoints

- **Desktop**: 1024px and above
- **Tablet**: 768px - 1023px
- **Mobile**: Below 768px
- **Small Mobile**: Below 576px
